# Tech Stack
- **Language**: Go
- **TUI Framework**: [Bubble Tea](https://github.com/charmbracelet/bubbletea)
- **AI Orchestration**: [Fantasy](https://charm.land/fantasy)
- **Database**: SQLite (via `sqlc` and `ncruces/go-sqlite3`)
- **CLI Framework**: [Cobra](https://github.com/spf13/cobra)
- **Shell Interpreter**: [mvdan/sh](https://github.com/mvdan/sh)
- **MCP Support**: stdio, http, sse
- **LSP Support**: gopls and other language servers
