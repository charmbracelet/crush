# VeniceCode Changelog

All notable changes to VeniceCode will be documented in this file.

## [1.0.0] - 2024-11-24

### Added
- Venice.ai provider integration with 10+ models
- Enhanced write tool with array content support (fixes GLM 4.6 / Qwen 3 Coder compatibility)
- Venice-specific documentation and guides
- Pre-configured Venice.ai models (Llama 3.3 70B, Qwen, Deepseek Coder, etc.)
- Installation scripts for multiple platforms
- Homebrew formula for easy macOS installation

### Changed
- Default provider set to Venice.ai
- Default model set to llama-3.3-70b
- README updated with Venice.ai branding

### Fixed
- Write tool now handles both string and array content formats
- Compatibility with GLM 4.6 and Qwen 3 Coder 480B models
