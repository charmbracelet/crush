# Use Cases Overview

Crush supports several key use cases for AI-assisted development.

## Core Use Cases
- [Interactive Coding Assistance](interactive-coding.md)
- [Project Initialization](project-initialization.md)
- [Multi-File Refactoring](refactoring.md)
- [External Tool Integration](mcp-integration.md)

## Summary Table

| Use Case | Trigger | Main Benefit |
|----------|---------|--------------|
| **Interactive Chat** | User prompt in TUI | Real-time assistance and problem solving. |
| **Code Search** | `grep`, `glob`, `sourcegraph` tools | Quickly find context in large codebases. |
| **Automated Edits** | `edit`, `multiedit` tools | Apply changes across one or many files safely. |
| **Build/Test** | `bash` tool | Run and verify code without leaving the chat. |
