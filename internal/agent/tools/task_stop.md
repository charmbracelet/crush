Requests cancellation of a task graph task.\n\nUse this tool to stop one task or all tasks in an active mailbox.\n\nParameters:\n- mailbox_id (required): Mailbox identifier (typically the parent task graph tool call ID).\n- task_id (optional): Specific task ID to stop. Omit to request stop for all tasks in that mailbox.\n- reason (optional): Human-readable cancellation reason.\n\nNotes:\n- Stop is cooperative and applied by the task graph runtime.\n- Unknown mailbox or task IDs return an error response.\n
